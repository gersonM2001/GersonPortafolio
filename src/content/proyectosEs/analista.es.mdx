# ¿Qué se realizó?
Este proyecto se inició de cero, usando una base de datos relacional, desarrollada en oracle, que tenia información acerca
de ventas de una empresa de venta de autorepuestos.

El proyecto requeria que se pasara de un modelo relacional, que es como la información se encontraba originalmente, y se la extrajera
de esa base, para luego transformarla en información sencilla y mas depurada, facilmente entendible, para colocarla en una nueva
base de datos, esta vez en *modelo copo de nieve*, pero en el gestor de *SQL Express*.
Este último para conectarlo con la herrmaienta *Tableu*, para generar dinámicamente, gráficos llamativos, y que fuera fáciles de comprender y explicar
como estaba funcionando el histórico.

## ¿Cómo se trabajó?
Se desarrollo a traves de una webapp, en el frontend con html, css y javascript; para la parte del backend 
se uso nodejs y express.
