# Objetivo
Swiftdrop, es una aplicación de delivery, similar a pedidos ya!, Hugo, etcétera, con la diferencia que esta aplicación buscaba
una relación cliente-vendedor-delivery, buscando que estos últimos tengan un mejor trato del que tienen actualmente;
por la parte de vendedores, tambien se buscaba que fuera una aplicación mas abierta, es decir que pequeñas pulperias, ferreterias
entre otras que no pueden contar con un conductor exclusivo, pudieran tener acceso a este servicio, mejorando así su competitividad de 
mercado.

## Alcance del proyecto

Este proyecto, se realizó haciendo uso únicamente de HTML, CSS, y JavaScript, por el lado del frontend.
En la parte del backend, se hizo uso de NodeJs, TypeScript y express; por último se optó por el uso de una base de datos
no relacional, en este caso, MongoDB
