# SharckCat 

Esta plataforma fue diseñada para ayudar a **estudiantes** de diferentes ramas de las ciencias, consiste en conectar personal capacitado (tutores) en distintas categorias, como Matemáticas, Programación, Artes, etcétera., con personas que tenga problemas con estas asignaturas (pupilos).

## Dirigido a:
- Personas no profesionales, pero que tenga conocimiento comprobable de las asignaturas que desea dar.
- Estudiantes que tengan conocimiento comprobable de laa asignaturas que desea dar y quiera obtener ingresos extras.
- Estudiantes que tengan dudas o consultas respecto a temáticas de distintas áreas del conocimiento.

## Tecnologías implementadas.

- NextJs
- NestJS
- MySql
- Railway
- Prisma
- AWS
